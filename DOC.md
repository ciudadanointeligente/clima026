## Sitio web y app

* Stack: Astro, Keystatic, React, Tailwind CSS, Alpine.js.

## Descripción

* Sitio web para mostrar la evaluación realizada a propuestas y/o medidas legislativas.

## Modelo de datos

* Categoría
  * Subcategoría
    * Criterio
Cada medida evaluada debe formar parte de al menos una subcategoría, la que la relaciona con una categoría en forma indirecta.

* Criterios de evaluación
  * Nombre del criterio y su explicación

Cada medida evaluada debe ser contrastada al menos con un criterio, pero puede ser sometida a varios.

* Criterios de evaluación: Están todas cruzadas por dos clasificaciones: 
  * Tipo 
  * Principio
